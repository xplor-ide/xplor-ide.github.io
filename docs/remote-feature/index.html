Remote Feature

What does your user need to know to use cloud service in desktop IDE?
Tags:
  • Test
  • Docs
Categories:
  • Examples
  • Placeholders

This section guides users on how to connect to the cloud service and utilize its features independently.

  • Connecting to the Cloud from the IDE

  • Key Feature: Remote Debugging and Usage Instructions

7.1 XploR Labs

XploR Labs is an extension that enables users to connect to their cloud workspaces and access them remotely from a local machine.

Step to connect:

  1. Click to XploR Labs icon in the Activity Bar.

  1. Select the login button and enter the cloud server URL. (The IDE supports all cloud systems based on Coder Server.)

  1. The IDE will open a browser to retrieve a session token for authentication. Copy the token and paste it into the IDE.

  1. Once logged in, all workspaces will be displayed in the XploR Labs widget tree view.

7.2 Remote connection to Workspace

To connect to a cloud workspace:

  1. Click the Open Workspace button in the tree view or the Open Workspace button in the bottom-right corner.

The IDE will connect to the cloud, set up the environment, and open a new window with remote access to your cloud workspace.
